Clipping Paths

Clipping paths restrict drawing to a particular area of the canvas. Think of clipping paths as mask - areas where the mask is empty show through, while areas where the mask is opaque to not Images can be drawn directly onto a canvas element in a variety of ways. Your code can drawn an image directly from an IMG or VIDEO element that is already in the page or loaded dynamically via script, or another canvas element can be used as a source image.
