Scalable Vector Graphics are vector based images that can adjust for different scales. They are helpful for creating content that will display effectively across different screen sizes. If you are trying to keep text looking sharp whether it is on widescreen computer monitor or an old iPhone, SVG can help. On the technical side SVGs work by displaying different images based upon what resolution the viewing screen is. So if the media screen is set at 400 pixels, below 400 pixels one image will display and above 400 pixels a different image will display.
